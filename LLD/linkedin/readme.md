### Requirements
- User should create and manage thier profiles
- User should have professional information like profile picture, name, headline, summary, experience, education, certification, skills and so on.
- User should send, accept/reject and view the connections and connection level like 1st, 2nd & 3rd+
- User should interact with others via chat.
- User should post their ideas, acheivements, research, finding, job openings and many more.
- User should search and apply for the job openings
- Recuriters can search candidate and reach out to them via mail.
- User should search people, companies and job postings based on different filters.
- User should receive notification for new connection request, message, job alerts
